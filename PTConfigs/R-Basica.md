1. Apagar el Router
2. Agregar Interfaces Hw

| Comando            | Descripción |
| -------            | ----------- |
| enable             | Privilegios |
| configure terminal |             |
| interface giga...  | Indicamos interfaz|
| ip address 192.0.0.1 255.255.255.0 | Direccionamos|
| no shutdown | Encendemos |
